# Audit AI Platform (audit.coach & audit.pro)

An AI-powered platform revolutionizing medical device auditing, quality management, and regulatory compliance.

## Core Features

- AI-Assisted Audit Checklist Generation
- Real-Time AI Scribe for Audit Documentation
- Regulatory Intelligence and Compliance Monitoring
- QMS Document Management and Review
- Continuous CRAID (Compliance, Risks, Assumptions, Issues, Dependencies) Monitoring
- Advanced Analytics and Reporting

## Project Structure

- `backend/`: Contains the API and database management code
- `frontend/`: Houses the React-based user interface
- `ai-services/`: Manages AI integrations and models
- `regulatory-db/`: Stores and manages regulatory data
- `docs/`: Contains project documentation
