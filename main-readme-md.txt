# Secure Tab Manager

Secure Tab Manager is a browser extension that allows users to manage tabs, create knowledge bases, and communicate between tabs securely.

## Features

- Secure tab management across multiple windows
- Knowledge base creation from tab contents
- Encrypted storage of sensitive information
- Real-time communication between tabs
- User authentication and authorization

## Project Structure

- `extension/`: Contains the browser extension code
- `backend/`: Contains the backend server code
- `docs/`: Contains user and developer documentation

## Quick Start

1. Clone the repository
2. Install dependencies: `npm run install:all`
3. Build the project: `npm run build`
4. Start the backend server: `npm run start:backend`
5. Load the extension in your browser (see [Developer Guide](docs/DEVELOPER_GUIDE.md))
